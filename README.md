# onion-cli

![Npm 版本](https://img.shields.io/badge/onion-cli_v0.0.3-green)

## 简介

用于快速搭建前端项目的命令行工具。

## 模版

- 快速生成 Vue3 + TS + Vite + Pinia + Axios + Mock 的模板

## 安装

```bash
npm install xzx-onion-cli -g
```

## 使用

```bash
 # 直接创建项目
onion create

 # 创建指定名称的项目
onion create my-app

# 查看脚手架版本
onion -v
或
onion --version


```
